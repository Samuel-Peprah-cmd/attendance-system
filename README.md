# ATOM Gate by AtomDev Studios is Security Attendance System

A Flask-based security portal for schools featuring QR scanning, 
GPS geofencing, and automated SMS alerts.

## Tech Stack
- Backend: Flask / SQLAlchemy
- Storage: Cloudflare R2
- Notifications: Twilio SMS & SendGrid Email
- Frontend: Tailwind CSS

## Deployment Notes
- Hosted on: PythonAnywhere
- Static Path: /home/YOURUSERNAME/mysite/app/static
